
=======================================
sdcard
=======================================

The sdcard folder provides limited pre-loaded
media content for use with the following Oculus
Sample Applications provided with this SDK:

- Oculus Cinema
- Oculus 360Photos
- Oculus 360Videos
- VrScene

To add additional media to the sdcard content,
please start by reviewing the Media Creation
Guidelines in the Mobile SDK Documentation to
verify content compatibility.

The table below provides the sdcard locations 
where additional media should be placed.

---------------------------------------------------------
Application   | Content Type | Media Path on sdcard
---------------------------------------------------------
Oculus Cinema |   2D Movie   | Movies\
	      |		     | DCIM\
	      |		     | Oculus\Movies\My Videos
---------------------------------------------------------
Oculus Cinema |   3D Movie   | Movies\3D\
	      |              | DCIM\3D\
	      |		     | Oculus\Movies\My Videos\3D\
----------------------------------------------------------
Oculus 360    |  360 degree  | Oculus\360Videos\
  Video	      |	 Panoramic   |
	      |  Video	     |
----------------------------------------------------------
Oculus 360    |  360 degree  | Oculus\360Photos\
  Photos      |  Static      |
	      |  Photos      |
----------------------------------------------------------
